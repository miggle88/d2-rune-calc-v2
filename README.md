# Diablo II Rune Calculator

A web app for tracking your Diablo II rune stash, calculating runeword possibilities, and planning Horadric Cube upgrades.

**Live App:** https://miggle88.github.io/d2-rune-calc-v2/

---

## Features

- **Rune Stash** — Track quantities for all 33 runes (El through Zod)
- **Runewords** — See which runewords you can make, which you're close to, and filter by slot type
- **Cube Upgrades** — See how many times you can upgrade each rune using Horadric Cube recipes
- **Cloud Sync** — Create an account to save your stash and access it from any device

---

## How to Use

### 1. Create an Account
Click **Sign Up**, enter your email and password, then confirm your email. Sign in to access your stash.

### 2. Track Your Runes
Enter the quantity of each rune you own by clicking the number field on any rune cell in the Rune Stash grid. Your stash saves automatically.

### 3. Check Runewords
The **Runewords** panel updates in real time:
- **Green** — You have all the runes needed to make this runeword
- **Yellow** — You're close, missing only 1–2 rune types
- **Dimmed** — You don't have enough runes yet

Use the **status filter** (All / Can Make / Close) and **slot type dropdown** to narrow down results.

### 4. Plan Cube Upgrades
The **Cube Upgrades** panel shows how many times you can upgrade each rune using the Horadric Cube. Runes from Pul onwards also require a Flawless Gem.

### 5. Send Feedback
Click the **Feedback** button in the top right to report a bug or request a feature.
